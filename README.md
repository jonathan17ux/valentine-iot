# Valentine's IoT Project

Cross-country emoji communication system connecting Chile and Miami via encrypted VPN.

## Tech Stack
- **Server:** Raspberry Pi 5, Ubuntu Server 24.04, WireGuard VPN
- **Clients:** Raspberry Pi Zero 2W, 480x320 TFT displays
- **Backend:** Python, Flask, SocketIO, SQLite3
- **Security:** UFW Firewall, fail2ban, WireGuard encryption
